BreakOut

by: Meijia He (Grace)

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decide to make this game because when I was a kid, I really like to play break out games. So as I saw this project, I am planning to make a special, unusual break out game. I choose to make this game rather than other games because it's concept is easier to make and much like the last project that I did.

2. What I Learned

I have learned many problems of my game from user testing because my classmates and teacher really gave me some good advice like menu state settings and frame problems in the android system computer and my game's functions problems.

3. Architecture of My Game

Different parts of my game interact by bouncing off each other or as the ball hits the blocks, that blocks disappear. Also, there is a paddle to make interactions with the ball after it bounces back. The overall architecture is like a wall half with balls and the rest is spaces to let the ball and paddle function.

4. Technical Implementation

My most proud of part in this game is that rather than normal breakout games, I have many functions blocks or special effect blocks during the game. Also, the framed picture and the game sound is also very nice. I have thinked to make some special effect blocks then I take steps in eclipse to make my blocks more interesting. For the frame picture and game voice, I have researched many of them finally decided the bests.

5. Most Challenging Part

The most challenging part of this project is my restart page of this game often goes wrong as I add some special effects blocks or adjust my game a bit, the restart state would go wrong.

6. Future Fixes and Updates

I would like to change my fonts to more pretty and also make more levels for my game.
